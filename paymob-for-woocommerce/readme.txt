== Description ==
= Introduction =
Paymob runs millions of transactions for different business sizes  across the Middle East and Africa. Paymob simplifies accepting online debit/credit payments for merchant's online stores.

Paymob set up a wide range of local and international payment options to maximize revenue generation opportunities such as credit and debit cards, online mobile wallets, cash on delivery, loyalty points redem options, payment links and even kiosk payments through Aman and Masary.

Paymob's card payments app provides secure, PCI-compliant  online debit/credit payments with 3DS protection while empowering you with an easy order management and analytical reports  to grow your business.

== Installation ==
You should have an active account with PayMob.

= Configuration =
1. Log into Paymob account => Setting in the left menu. Then get the Secret, public ,and API keys.
2. Paste each key in its place in the Paymob WooCommerce setting page.
3. Then, click on Validate PayMob API key button to ensure the data prvided keys are valid and returns the needed information. (Important)
4. The HMAC key will be shown in its field and the existing Paymob integration IDs as well. In case any of the provided data is not correct, the validation will fail.
5. Please ensure to select the integration IDs that you need end-user pay with/sees in Paymob payment page. (Important) 
6. Copy integration callback URL that exists in Paymob WooCommerce setting page. Then, paste it into each payment integration in Paymob account
